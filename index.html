🚀 Quick Start Guide

This version integrates with Google Sheets API to fetch your location data in real-time!

⚙️ Configuration Settings

✨ Easiest Method - No API Needed!

  1. Open your Google Sheet
  2. Click File → Share → Publish to web
  3. Select Entire Document and CSV format
  4. Click Publish and confirm
  5. Enter just your Sheet ID below - no API key needed!

📋 Alternative: Use Google Sheets API

  1. Enable Google Sheets API in Cloud Console
  2. Add Sheets API permission to your API key
  3. Make sheet viewable by "Anyone with link"
Find this in: docs.google.com/spreadsheets/d/[SHEET_ID]/edit
Leave default if your data starts at A1
Skip this if you published your sheet to web
